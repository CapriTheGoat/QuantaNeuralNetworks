# Quanta Neural Networks

**Author:** Riccardo Caprioglio.

**Work:** MSc thesis for Computer Science and Engineering Degree at Politecnico di Milano.

**Progress:** As of now added a folder containing files for baseline classification of [SPAD-MNIST images](https://boracchi.faculty.polimi.it/Projects/SPAD-MNIST.html). Folder contains: baseline classification pipeline using original wision-lab integrator and SSD script, modified dataloader for simulated SPAD-MNIST dataset, train and eval script for the SPAD-MNIST dataset.
